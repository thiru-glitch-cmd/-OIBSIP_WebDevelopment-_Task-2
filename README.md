# Task 2: Personal Portfolio Website

A clean, responsive, and modern personal portfolio website built with HTML and CSS to showcase skills, certifications, projects, and contact details.

## Objective
To design and develop an interactive personal portfolio website that is visually appealing, professional, and fully responsive across all device screen sizes (mobile, tablet, desktop).

## Steps Performed
1. **HTML Structure**: Created a semantic layout using HTML5 tags including `<header>`, `<nav>`, `<section>`, and `<footer>` to define logical divisions for Home, About, Skills, Certifications, Projects, and Contact sections.
2. **Styling & Theme**: Styled the layout using Vanilla CSS, establishing a sleek, premium dark-theme (`#0f172a` slate background) accented with vivid sky-blue (`#38bdf8`) highlighting.
3. **Interactive UI Elements**:
   - Integrated hover effects and transitions (`transition: 0.3s`) for links, buttons, and cards.
   - Built a dynamic translation animation (`transform: translateY(-8px)`) on hover for project, skill, and certification cards to improve user engagement.
4. **Grid-Based Layouts**: Utilized CSS Grid with `repeat(auto-fit, minmax(...))` to automatically wrap cards responsively without requiring complex breakpoints.
5. **Responsiveness**: Designed a mobile-friendly view using CSS Media Queries, collapsing navigation and sizing down typography appropriately for small screen widths.

## Tools Used
* **Core Languages**: HTML5, CSS3 (Vanilla CSS)
* **Typography**: Google Fonts (Poppins)
* **IDE & Editors**: VS Code / Antigravity Agentic Environment

## Outcome
A professional, production-ready single-page portfolio website that effectively presents an individual's background, skills, certifications, and project links. The website features fluid transitions and a clean, responsive layout across all display screens.
